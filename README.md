# SpheArt

技術に関する記事の投稿・閲覧ができるQiitaとZennというサービスで投稿された記事を集約するサービスです。

### 現在提供している機能
- 記事の閲覧
  - 記事カードを押下すると該当の記事ページへ遷移します（外部リンク）
- 記事の検索
  - 記事名に部分一致する文字列を用いて検索ができます
- サインアップ/ログイン/ログアウト機能
  - ID/Passを用いて認証を行います
- お気に入り機能（ログイン時）
  - 記事をお気に入り登録できます
